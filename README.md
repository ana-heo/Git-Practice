# Git Practical Task

Git hands-on practice:
- Creating and switching branches
- Working with files in different branches
- Performing merges
- Analyzing commit history

  As part of the task was:

- Created a separate branch called "develop"
- Added test files to different branches (main and "develop")
- Made commits describing the changes
- Merged the "develop" branch into "main"
- Analyzed the commit history
  

  Commands used:

- git branch 
- git checkout
- git commit -m
- git merge
  

develop_test.txt - File created and modified in the develop branch,

main_test.txt - File related to the main branch, 

command_list.csv - List of used Git commands, 

.gitignore - File of ignored Git objects.


# Result

- understanding the Git branching model
- the difference between the main and working branches
- the process of merging changes
- working with commit history
